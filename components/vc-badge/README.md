
## API

### Badge【props】

| 属性 | 描述 |
| --- | --- |
| value | [可选] 设置数值，优先级会高于标签内嵌套值 |
| max | [可选] 设置最大边界，越界数值显示为 maxVal+ |
| type | [可选] 设置标记类型，职位 dot 显示为弱提示的圆点 |
