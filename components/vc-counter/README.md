### Counter Props

| 名称                  | 描述                         |类型
|----------------------|------------------------------|--------|
|`value`           |  默认值1  |Number
|`maxValue`        |  最大value，默认10000  | Number
|`minValue`        | 最小value，默认1 | Number
|`onChange`        | value改变触发事件| 
|`disabled`        | 是否禁用|Boolean
